# C1Reaction
Simple medidor de velocidad de reacción basado en Arduino

### Instalación

Sólo necesitas [esta libraría](https://github.com/fdebrabander/Arduino-LiquidCrystal-I2C-library).

### Conexiones

| Arduino |    Peripherial   |
|:-------:|:----------------:|
| A4      |     SDA (LCD)    |
| A5      |     SCL (LCD)    |
| D2      | Interrupt switch |
| D3      |      Buzzer      |
| D4      |   On/Off swicth  |
